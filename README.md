# AI Team Allocation System

AI-driven optimization system for automated fair team distribution using Simulated Annealing, statistical balancing, and multi-dimensional scoring.

---

## Problem
Manual team selection often leads to bias and unbalanced teams.

---

## Solution
This system automatically generates balanced teams using optimization algorithms and statistical evaluation.

---

## Features
- Automated team generation
- Multi-dimensional scoring
- Simulated Annealing optimization
- Statistical balance evaluation
- Fair distribution across teams

---

## Input
- Participant ID
- Multiple evaluation scores

---

## Output
- Balanced teams
- Score distribution per team
- Balance metrics (min/max/difference)

---

## Algorithm
- Simulated Annealing
- Random swap optimization
- Objective function minimizing variance

---

## Tech Stack
- Python
- Statistics
- Optimization algorithms

---

## How it works
1. Load data
2. Initialize random teams
3. Compute balance score
4. Swap participants randomly
5. Accept/reject based on optimization rule
6. Return best configuration

---

## Results
- Reduced team imbalance
- Improved fairness
- Stable reproducible allocations
